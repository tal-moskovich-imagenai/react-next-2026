## React is a rendering paradigm

<v-clicks>

- Not just a web framework
- Same component model · same hooks · same Flexbox
- Just a **different output target**

</v-clicks>

<v-click>

```
Your Components (JSX, hooks, state)
        ↓
  The Reconciler (what changed?)
        ↓
  The Renderer  (how to apply it)
```

</v-click>

<v-click>

> Layer 3 is **pluggable**.
>
> `react-dom` → Browser DOM
> `react-native` → Mobile widgets
> **`ink`** → Terminal output

</v-click>

::right::
